---

## 内容规范

| 页面 | 核心元素 | 规则 |
|------|---------|------|
| 封面 | 主标题2行 | 第2行使用强调色 |
| 内页 | 清单项目 | 3-5条，每条包含标题+提示 |
| 内页 | 互动投票 | 底部投票区，问题+选项 |
| 结尾 | 结语金句 | 3-5行短句，居中排列 |
| 结尾 | CTA引导 | 引导评论/点赞/收藏 |

---

## 清单项目格式

```javascript
// 单个清单项
{
  "title": "清单项标题（≤20字）",
  "hint": "→ 提示语（≤15字）"
}
```

- 每页 3-5 条清单项
- 标题简洁有力，动词开头
- 提示语以"→"开头，解释原因或好处

---

## 互动区格式

```javascript
{
  "emoji": "🎯",           // 1个emoji
  "label": "今日投票",      // 2-4字标签
  "question": "问题？",     // ≤20字
  "options": ["选项1", ...]  // 2-4个选项
}
```

---

## 配色

| 用途 | 色值 |
|------|------|
| 背景主色 | #0f1419 |
| 背景渐变 | #1a1f2e → #121620 |
| 强调色（柠檬黄） | #FFEF00 |
| 强调色渐变 | #FFF44F → #FFD700 |
| 主文字 | #ffffff |
| 次要文字 | #ffffff (opacity 0.6-0.8) |

---

## 检查清单

- [ ] 封面标题分2行，第2行黄色
- [ ] 内页3-5条清单项
- [ ] 每条清单有标题+提示
- [ ] 底部有互动投票区
- [ ] 结尾页有CTA引导
- [ ] 页码格式 "01", "02", "03"
