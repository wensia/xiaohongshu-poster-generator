# 性格独白风 - 核心规则

## 数据格式

### 封面 (cover)
```json
{
  "zodiac": "射手座",
  "topic": "性格独白",
  "subtitle": "唯独受不了委屈",
  "titleLine1": "什么苦都能吃",
  "titleLine2": "却扛不住心寒",
  "tagline1": "看起来没心没肺",
  "tagline2Text": "其实",
  "tagline2Highlight": "心思细腻",
  "pageNum": "0 1"
}
```

### 内页 (page)
```json
{
  "zodiac": "射手座",
  "topic": "性格独白",
  "partNum": "01",
  "sectionTitle": "表面乐观",
  "content": [
    "射手表面乐观",
    { "before": "其实内心", "highlight": "敏感", "after": "得要命" },

    { "before": "习惯了", "highlight": "自己扛", "after": "" }
  ],
  "quote": "走得洒脱，是因为早就做好了准备",
  "pageNum": "0 2"
}
```

### 结尾页 (summary)
```json
{
  "zodiac": "射手座",

  "endingLine2": "心中有梦",
  "pageNum": "0 8"
}
```

---

## 内容规范

| 页面 | 高亮词数量 | 规则 |
|------|-----------|------|
| 封面 | 2个 | 主标题1个 + 标语1个 |
| 内页 | 2个 | 形成语义呼应对（因果/对比/递进/转折） |
| 结尾页 | 1-2个 | 与封面形成首尾呼应 |

---

## 正文规范

- **行数**：3-7行短句
- **每行**：≤22个中文字符
- **行间距**：61px
- **引用金句**：8-20字，用引号包裹

---

## 高亮词格式

```javascript
// 纯文本
"射手表面乐观"

// 带高亮
{ "before": "其实内心", "highlight": "敏感", "after": "得要命" }
```
