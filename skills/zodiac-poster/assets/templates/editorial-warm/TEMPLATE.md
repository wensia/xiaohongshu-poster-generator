# 编辑暖调 (Editorial Warm) 模板规范

## 画布尺寸

**尺寸**：1080px × 1440px（3:4 比例）

---

## 设计系统

### 配色规范

| 用途 | 色值 | 说明 |
|------|------|------|
| 重点色 | `#C15F3C` | Claude VI Crail，赭红/珊瑚橙 |
| 主文字 | `#2D2D2D` | 深灰，用于标题 |
| 正文字 | `#3D3D3D` | 次深灰，用于段落 |
| 辅助文字 | `#5A5A5A` | 中灰，用于副标题/引用 |
| 弱化文字 | `#B1ADA1` | Claude VI Cloudy，用于标签/页码 |
| 装饰线 | `#D9D6D0` | 浅灰，用于分隔线 |

### 背景渐变

```css
.poster {
  background: linear-gradient(
    165deg,              /* 渐变角度：左上到右下偏斜 */
    #FAF6F1 0%,         /* 起点：最浅 */
    #F8F2EB 30%,        /* 30%位置：略深 */
    #F5EDE4 60%,        /* 60%位置：次背景色 */
    #F2E8DD 100%        /* 终点：最深 */
  );
}
```

**渐变规则：**

| 规则 | 说明 |
|------|------|
| 色相 | 统一暖黄/米色调 |
| 渐变方向 | 165° 斜向（左上浅 → 右下深） |
| 明度变化 | 从亮到暗，差距约 2-3% |
| 色值规律 | R 值固定 FA-F2，G/B 递减 |

**色值递进：**
```
#FAF6F1 → #F8F2EB → #F5EDE4 → #F2E8DD
   ↓          ↓          ↓          ↓
 最浅      稍深       次深       最深
```

### 重点色使用原则

重点色 `#C15F3C` 应用于以下元素，确保视觉层次：

1. **必用** — 星座线性图标
2. **必用** — 关键词/标签文字（如「暧昧期指南」）
3. **必用** — 核心强调词（如「射手回避型」「喜欢」「躲压力」）
4. **必用** — 双引号装饰
5. **可选** — 装饰横线（标签两侧）
6. **可选** — 引用区块左边框（使用 40% 透明度）

### 字体规范

```css
font-family: 'Noto Serif SC', serif;
```

- 字体来源：Google Fonts
- CDN 引用：`https://fonts.googleapis.com/css2?family=Noto+Serif+SC:wght@400;500;600;700&display=swap`

| 元素 | 字号 | 字重 | 字距 |
|------|------|------|------|
| 主标题 | 84px | 400 | 8px |
| 副标题 | 52px | 400 | 4px |
| 正文 | 34px | 400 | 2px |
| 关键词 | 28px | 400 | 8px |
| 标签/页码 | 24px | 400 | 4px |

### 纸张纹理

```css
.poster::before {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  background-image: url("data:image/svg+xml,%3Csvg viewBox='0 0 200 200' xmlns='http://www.w3.org/2000/svg'%3E%3Cfilter id='noise'%3E%3CfeTurbulence type='fractalNoise' baseFrequency='0.8' numOctaves='4' stitchTiles='stitch'/%3E%3C/filter%3E%3Crect width='100%25' height='100%25' filter='url(%23noise)'/%3E%3C/svg%3E");
  opacity: 0.06;
  pointer-events: none;
}
```

- `baseFrequency`: 0.8（颗粒密度）
- `opacity`: 0.06（不干扰阅读）

## 封面模板结构

**尺寸**：1080px × 1440px
**内边距**：120px 100px
**布局**：居中对齐

### 结构层级

```
┌─────────────────────────────────┐
│ [标签]              [星座图标] │  ← header
│                                 │
│                                 │
│         ── 关键词 ──            │
│           主标题                │  ← main（居中）
│         副标题/强调             │
│          点睛文案               │
│                                 │
│                                 │
│ ─────────────────────────── 01 │  ← footer
└─────────────────────────────────┘
```

### 元素样式

| 元素 | 颜色 | 说明 |
|------|------|------|
| 左上标签 | `#B1ADA1` | 如「射手座 · 回避型」 |
| 右上图标 | `#C15F3C` | 星座线性SVG，64px |
| 关键词 | `#C15F3C` | 带两侧横线装饰（横线48px） |
| 主标题 | `#2D2D2D` | 最大字号，视觉焦点 |
| 副标题 | `#5A5A5A` | 含重点色强调词 |
| 点睛文案 | `#B1ADA1` | 一句话总结 |
| 底部线 | `#D9D6D0` | 贯穿底部 |
| 页码 | `#B1ADA1` | 右下角，如「01」 |

## 星座图标 SVG

**图标容器**：64px × 64px

射手座（♐）线性图标：

```html
<svg viewBox="0 0 50 50" width="64" height="64">
  <line x1="8" y1="42" x2="42" y2="8" stroke-linecap="round"/>
  <polyline points="30,8 42,8 42,20" stroke-linecap="round" stroke-linejoin="round"/>
  <line x1="18" y1="32" x2="32" y2="18" stroke-linecap="round"/>
  <line x1="12" y1="28" x2="22" y2="38" stroke-linecap="round"/>
</svg>
```

样式：
```css
stroke: #C15F3C;
stroke-width: 1.5;
fill: none;
```

其他星座需替换为对应符号的线性设计。

## 内容填充指南

### 封面文案结构

```
标签：[星座] · [主题类型]
关键词：[主题分类]
主标题：[核心观点/钩子]
副标题：[星座] + [关键行为/特征]
点睛文案：[一句话总结/金句]
```

### 正文文案结构

```
标签：[星座] · [主题]
星座符号：Unicode 符号
星座名称：中文名称
正文：3-5 段，每段 1-2 句
强调词：用 <span class="highlight"> 标记
```
