## 内容规范

| 页面 | 核心元素 | 规则 |
|------|---------|------|
| 封面 | 主标题2行 | 第2行使用霓虹渐变 |
| 内页 | 弹幕气泡 | 5-7条，错落排列 |
| 内页 | 高亮弹幕 | 1条特殊样式，居中突出 |
| 内页 | 时间戳 | 每条弹幕带时间 |
| 结尾 | 结语金句 | 3-5行，居中排列 |

---

## 弹幕格式

```javascript
// 普通弹幕
{
  "text": "弹幕内容（≤20字）",
  "time": "23:47",
  "color": "purple" // purple/cyan/pink/teal/green
}

// 高亮弹幕（居中突出）
{
  "text": "重点弹幕内容（≤25字）",
  "time": "02:00",
  "highlight": true
}
```

### 弹幕颜色对应

| color | 边框色 |
|-------|--------|
| purple | #8B5CF6 |
| cyan | #06B6D4 |
| pink | #A855F7 |
| teal | #22D3EE |
| green | #10B981 |
