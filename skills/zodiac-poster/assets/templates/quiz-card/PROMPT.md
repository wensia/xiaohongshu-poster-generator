## 内容规范

| 页面 | 核心元素 | 规则 |
|------|---------|------|
| 封面 | 主标题2行 | 简洁有趣，引发好奇 |
| 封面 | 底部预览 | 显示A/B选项卡片预览 |
| 内页 | 问题题目 | 2行标题 + 副标题框 |
| 内页 | 选项卡片 | A/B两个选项，不同颜色 |
| 内页 | 结果区域 | 黑色卡片，显示答案解析 |
| 结尾 | 结果卡片 | 白色大卡片，完整结果 |
| 结尾 | 特质标签 | 3-5个彩色标签 |

---

## 选项格式

```javascript
// 选项卡片
{
  "label": "A",          // 选项标识
  "color": "#FF7043",    // 橙色为A，青色为B
  "text": "选项文字内容（≤18字）",
  "x": 100,              // 卡片X位置
  "y": 580               // 卡片Y位置
}
```

### 选项颜色对应

| 选项 | 颜色 | 色值 |
|------|------|------|
| A | 橙色 | #FF7043 |
| B | 青绿 | #00BFA5 |
| C | 黄色 | #FFE135 |
| D | 荧光绿 | #C6FF00 |

## 检查清单

- [ ] 封面有A/B选项卡片预览
- [ ] 内页问题清晰，选项对比明显
- [ ] 选项卡片使用不同颜色区分
- [ ] 结果区域使用黑色背景突出
- [ ] 结尾页有完整测试结果
- [ ] 底部有互动引导（分享/评论）
- [ ] 装饰形状丰富多彩
