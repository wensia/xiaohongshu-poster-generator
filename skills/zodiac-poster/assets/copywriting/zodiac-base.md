# 星座小红书文案：通用规则

本文档定义所有星座共享的文案规则，包括改写 Prompt、审核 Prompt、通用高危词等。

---

## 1. 输入变量定义

| 变量 | 说明 | 示例 |
|------|------|------|
| `{{zodiac}}` | 星座中文名 | 射手座 |
| `{{zodiac_en}}` | 星座英文名 | SAGITTARIUS |
| `{{topic}}` | 主题 | 轻计划学习法 |
| `{{pain_point}}` | 用户痛点 | 计划总是坚持不了 |
| `{{insight}}` | 核心洞察 | 有趣比系统重要 |
| `{{content}}` | 待改写的原文 | （用于改写场景） |

---

## 2. 改写 Prompt（通用）

```
你是一个小红书星座文案改写专家。请根据以下规则改写文案：

【星座人设】
请参考 {{zodiac}} 的人设配置文件

【改写要求】
1. 保持原文核心观点不变
2. 调整语气符合 {{zodiac}} 的说话方式
3. 替换高危词为推荐表达
4. 检查段落结构是否符合规范

【原文】
{{content}}

【输出格式】
- 改写后的文案
- 修改说明（列出主要调整点）
```

---

## 3. 审核 Prompt（通用）

```
你是一个小红书星座文案审核专家。请根据以下清单审核文案：

【审核清单】

□ 语气检查
  - 是否符合 {{zodiac}} 的人设？
  - 是否使用了禁用开场白？（姐妹们/宝子们集合啦）
  - 是否使用了禁用结尾？（点赞收藏不迷路）
  - 语气是否过于兴奋或过于消极？

□ 结构检查
  - 标题长度：10-20字
  - 正文段落数：6段左右
  - 每段长度：30-60字
  - 总字数：200-300字

□ 高危词检查
  - 是否包含通用禁用词？
  - 是否包含 {{zodiac}} 专属禁用词？

□ 内容检查
  - 是否有"首先...其次...最后"等列举式表达？
  - 是否有"总的来说/综上所述"等总结式表达？
  - 是否有过度使用感叹号？

【待审核文案】
{{content}}

【输出格式】
- 审核结果：✅ 通过 / ❌ 不通过
- 问题列表（如有）
- 修改建议
```

## 4. 通用高危词

### 禁用开场白

| 禁用 | 原因 |
|------|------|
| 姐妹们/宝子们集合啦！ | 过于热情，不符合冷淡调性 |
| 今天来聊聊... | 过于平淡 |
| 必看！必收藏！ | 过于营销感 |
| 任何带多个感叹号的开场 | 情绪过于激动 |

### 禁用结尾

| 禁用 | 原因 |
|------|------|
| 点赞收藏不迷路！ | 过于营销 |
| 关注我看更多！ | 过于营销 |
| 评论区告诉我！ | 过于套路 |

### 禁用表达

| 禁用 | 替换为 |
|------|--------|
| 宝子们/姐妹们 | 我们{{zodiac}}啊 |
| 超级无敌/爱死了 | 真的挺/确实 |
| 太难过了呜呜 | 心里不好受 |
| 说走就走超飒 | 转身就走 |
| 太真实了！ | 懂的都懂 |
| 笑死我了哈哈哈哈 | 想想还挺好笑 |
| 真的真的超级无敌 | 真的挺/确实 |
| 首先...其次...最后... | （重新组织，用短句） |
| 总的来说.../综上所述... | （直接删除或用"说白了"） |

### 推荐表达

| 场景 | 推荐用语 |
|------|----------|
| 表达喜欢 | 在意、放在心上 |
| 表达难过 | 心里不好受、默默咽了 |
| 表达果断 | 转身就走、不回头 |
| 表达认同 | 懂的都懂 |

## 5. 段落结构规范

### 6段式结构

| 段落 | 角色 | 句数 | 说明 |
|------|------|------|------|
| 第1段 | 开篇点题 | 1-2句 | 直入主题，不废话 |
| 第2段 | 核心观点 | 1-2句 | 抛出核心论点 |
| 第3段 | 维度A | 1-2句 | 第一个展开维度 |
| 第4段 | 维度B | 1-2句 | 第二个展开维度 |
| 第5段 | 维度C | 1-2句 | 第三个展开维度（可选） |
| 第6段 | 收尾 | 1-2句 | 留有余韵，不强求互动 |

### 句式特点

- **短句为主**：心里有本账，默默给你打分。
- **善用破折号**：转身比谁都快——不是不爱，只是尊严比感情更重要。
- **适度省略号**：那些回忆啊，就当是做了场醒不来的梦吧。

### 去AI痕迹技巧

| 技巧 | 示例 |
|------|------|
| 加入口语化表达 | emmm怎么说呢、好像是这样、反正就是 |
| 加入第一人称 | 我们{{zodiac}}啊、说实话、其实早就 |
| 不规则标点 | 用破折号代替逗号、偶尔省略句号、用省略号表示停顿 |

## 6. 模板参考

### Cool型（推荐）

```
{{zodiac}}的{{topic}}，从来不是{{常见误解}}。

{{核心表现1}}。{{核心表现2}}，{{转折或递进}}。

{{情感升华或留白}}。
```

### 自述型

```
我们{{zodiac}}啊，{{核心特点}}。

{{具体表现1}}。{{具体表现2}}——{{解释或反转}}。

{{收尾感慨}}。
```

### 观察型

```
{{zodiac}}{{行为/特点}}，{{简短描述}}。

{{展开1}}；{{展开2}}；{{展开3}}。

{{淡淡收尾}}。
```
