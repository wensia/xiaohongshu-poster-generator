## SVG 模板特点

- **纯 SVG 格式**：不再使用 HTML/CSS
- **Google Fonts CDN**：使用 `@import` 加载中文字体
- **渐变背景**：使用 `<linearGradient>` 定义
- **文字高亮**：使用 `<tspan fill="#C4653A">` 实现
