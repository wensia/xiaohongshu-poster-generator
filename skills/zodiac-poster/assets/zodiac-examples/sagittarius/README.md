## 适配模板

射手座（自由洒脱型）适配以下 5 套模板：

1. **动态编辑风** (editorial-dynamic) - 默认
2. **杂志分栏风** (magazine-part)
3. **极简留白** (minimal-whitespace)
4. **性格独白风** (personality-monologue) - 当前示例
5. **编辑暖调** (editorial-warm)
