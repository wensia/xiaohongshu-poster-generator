# 小红书文案生成器

从飞书多维表格拉取记录内容，生成符合小红书风格的文案。

⚠️ **回写要求（必做）**

生成后必须写回「星座海报生成」数据表：
- **标题**：写入封面标题
- **正文内容**：写入正文文案
（不要写到「低粉爆文抓取」表）

---

## ⚠️ 核心风格要求

### 整体调性：清冷但有温度

```
✅ 正确：淡淡的、娓娓道来、像在自言自语
❌ 错误：浮夸的、激动的、堆砌网络热词
```

### 风格定位

| 要素 | 要求 | 示例 |
|------|------|------|
| 人味 | 有，但不刻意 | "我们射手座啊" 而非 "射手宝宝们集合！" |
| 亲昵 | 有，但不轻浮 | "你遇到的是哪一种" 而非 "姐妹们快来看！" |
| 情感 | 克制，不煽情 | "心里有本账" 而非 "太扎心了呜呜呜" |
| 语气 | 平静，有力量 | "转身比谁都快" 而非 "说走就走超酷的！" |

---

## 文案生成规则

### 0. 标题要求

- **标题长度**：10-20 个字
- **标题风格**：简短、克制，不用感叹号

### 1. 字数要求

- **目标**：150-200字
- **段落**：5段，每段1-2句
- **每段字数**：50-80字
- **节奏**：短句为主，偶尔长句

### 2. 开头方式（选一）

| 类型 | 示例 |
|------|------|
| 自我代入 | "我们射手座啊，..." |
| 直接陈述 | "射手的感情，从来不是非黑即白。" |
| 淡淡感慨 | "说实话，..." / "其实..." |

**禁止使用的开头：**
```
❌ "姐妹们/宝子们集合啦！"
❌ "今天来聊聊..."
❌ "必看！必收藏！"
❌ 任何带感叹号的热情开场
```

### 3. 内容结构

```
第一段：点题 / 核心观点
   └── 1-2句，直接切入

第二段：展开 / 具体表现
   └── 2-3句，有细节但不啰嗦

第三段：收尾 / 情感升华（可选）
   └── 1-2句，留有余韵
```

### 4. 结尾方式（选一）

| 类型 | 示例 |
|------|------|
| 提问留白 | "你遇到的，是哪一种？" |
| 淡淡总结 | "有些人是过客，有些人才是归途。" |
| 自嘲收尾 | "没错，说的就是我自己。" |

**禁止使用的结尾：**
```
❌ "点赞收藏不迷路！"
❌ "关注我看更多！"
❌ "评论区告诉我！"
```

## 语言风格指南

### 推荐用词

| 场景 | 推荐 | 避免 |
|------|------|------|
| 表达喜欢 | "在意"、"放在心上" | "超爱"、"爱死了" |
| 表达难过 | "心里不好受"、"默默咽了" | "太难过了呜呜" |
| 表达决绝 | "转身就走"、"不回头" | "说走就走超飒" |
| 表达认同 | "懂的都懂" | "太真实了！" |

### 句式特点

```
✅ 用短句：
   "心里有本账，默默给你打分。"

✅ 用破折号：
   "转身比谁都快——不是不爱，只是尊严比感情更重要。"

✅ 用省略：
   "那些回忆啊，就当是做了场醒不来的梦吧。"

❌ 避免：
   "真的真的超级无敌..."
   "笑死我了哈哈哈哈"
   过多感叹号！！！
```

## 防AI检测技巧

### 1. 增加不完美感

```
✅ "emmm怎么说呢" → 口语化犹豫
✅ "好像是这样" → 不确定语气
✅ "反正就是" → 随意收尾
```

### 2. 加入主观视角

```
✅ "我们射手座啊"
✅ "说实话"
✅ "其实早就"
```

### 3. 使用不规则标点

```
✅ 破折号代替逗号
✅ 适当省略句号
✅ 偶尔用"..."留白
```

### 4. 避免AI腔调

```
❌ "首先...其次...最后..."
❌ "总的来说..."
❌ "综上所述..."
❌ 过于工整的排比
```

## 文案模板

### 模板 A：清冷型（推荐）

```
{星座}的{主题}，从来不是{常见误解}。

{核心表现1}。{核心表现2}，{转折或递进}。

{情感升华或留白}。
```

**示例：**
```
射手的感情，从来不是非黑即白。

天蝎是劫，爱得深也伤得重。双子是知己，懂你的自由，陪你一起疯。天秤是细水长流的舒服，不浓烈，但刚刚好。

有些人注定是过客，有些人才是归途。你遇到的，是哪一种？
```

### 模板 B：自述型

```
我们{星座}啊，{核心特点}。

{具体表现1}。{具体表现2}——{解释或反转}。

{收尾感慨}。
```

**示例：**
```
我们射手座啊，爱起来是火，走的时候是冰。心里有本账，默默给你打分，底线是最后那道坎，碰了就再也回不去。

别觉得我们没心没肺，其实早就把委屈悄悄咽了。一旦发现不被珍惜，转身比谁都快——不是不爱，只是尊严比感情更重要。

最后会安安静静地消失，不吵不闹，把体面留给自己。
```

### 模板 C：观察型

```
{星座}{行为/特点}，{简短描述}。

{展开1}；{展开2}；{展开3}。

{淡淡收尾}。
```

**示例：**
```
射手黑化起来，比谁都狠。

之前有多好说话，之后就有多绝情；以前看不透的人心，现在一眼看穿；曾经懒得用的心机，现在玩得比谁都溜。

别把善良当软弱，别把沉默当好欺。
```

## 执行流程

```
1. 从飞书拉取记录
   ├── 获取标题、正文内容、星座
   └── 记录 record_id

2. 分析内容主题
   ├── 配对类 / 性格类 / 情感类 / 黑化类
   └── 确定文案调性

3. 选择合适模板
   ├── 清冷型（大多数情况）
   ├── 自述型（第一人称内容）
   └── 观察型（分析类内容）

4. 生成文案
   ├── 遵循风格要求
   ├── 控制在 150-200 字
   └── 检查是否有AI腔调

5. 写回飞书（必须）
   ├── 标题字段 = 封面标题
   └── 正文内容字段 = 正文文案

6. 输出文案
   └── 可直接用于小红书发布
```

## 质量检查清单

生成文案后，检查以下项目：

- [ ] 标题长度 10-20 字
- [ ] 字数在 150-200 字之间
- [ ] 段落为 5 段
- [ ] 每段字数 50-80 字
- [ ] 没有使用感叹号堆砌
- [ ] 没有"宝子们"、"姐妹们"等浮夸称呼
- [ ] 没有"点赞收藏"等引导语
- [ ] 句式有变化，不是纯排比
- [ ] 有适当的破折号或省略号
- [ ] 结尾留有余韵，不是硬收
- [ ] 读起来像真人写的，不像AI
