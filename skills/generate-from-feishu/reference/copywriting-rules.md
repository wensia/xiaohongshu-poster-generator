# Copywriting Rules for Xiaohongshu

This document defines the style guidelines for generating Xiaohongshu captions and poster content.

---

## Core Style: Cool but Warm

```
✅ Correct: Understated, conversational, like talking to yourself
❌ Wrong: Exaggerated, excited, stuffed with internet slang
```

### Style Positioning

| Element | Requirement | Example |
|---------|-------------|---------|
| Human touch | Yes, but natural | "我们射手座啊" NOT "射手宝宝们集合！" |
| Intimacy | Yes, but not frivolous | "你遇到的是哪一种" NOT "姐妹们快来看！" |
| Emotion | Restrained, not melodramatic | "心里有本账" NOT "太扎心了呜呜呜" |
| Tone | Calm, powerful | "转身比谁都快" NOT "说走就走超酷的！" |

---

## Title Requirements

- **Length**: 10-20 Chinese characters
- **Style**: Short, restrained, no exclamation marks
- **Xiaohongshu limit**: Max 20 characters

---

## Content Requirements

### Word Count

| Property | Value |
|----------|-------|
| Target length | 200-300 characters |
| Minimum paragraphs | **6 paragraphs** (for 5 content pages) |
| Per paragraph | 30-60 characters |
| Sentence style | Short sentences primary, occasional long |

> **Why 6 paragraphs?**
> A poster set needs: 1 cover + 5 content pages. Each content page uses 1 paragraph.
> Plus 1 intro paragraph = **6 total minimum**.

---

## Opening Styles (Choose One)

| Type | Example |
|------|---------|
| Self-identification | "我们射手座啊，..." |
| Direct statement | "射手的感情，从来不是非黑即白。" |
| Soft reflection | "说实话，..." / "其实..." |

### Prohibited Openings

```
❌ "姐妹们/宝子们集合啦！"
❌ "今天来聊聊..."
❌ "必看！必收藏！"
❌ Any excited opening with exclamation marks
```

---

## 6-Paragraph Structure

```
Paragraph 1: Topic Introduction
   └── 1-2 sentences, get to the point

Paragraph 2: Core Viewpoint
   └── 1-2 sentences, main argument

Paragraph 3: Specific Aspect A
   └── 1-2 sentences, first dimension

Paragraph 4: Specific Aspect B
   └── 1-2 sentences, second dimension

Paragraph 5: Specific Aspect C (optional)
   └── 1-2 sentences, third dimension

Paragraph 6: Closing
   └── 1-2 sentences, leave lingering impression
```

Each paragraph should stand alone as content for one image.

---

## Ending Styles (Choose One)

| Type | Example |
|------|---------|
| Open question | "你遇到的，是哪一种？" |
| Soft summary | "有些人是过客，有些人才是归途。" |
| Self-deprecating | "没错，说的就是我自己。" |

### Prohibited Endings

```
❌ "点赞收藏不迷路！"
❌ "关注我看更多！"
❌ "评论区告诉我！"
```

## Language Style Guide

### Recommended Words

| Scenario | Recommend | Avoid |
|----------|-----------|-------|
| Expressing like | "在意"、"放在心上" | "超爱"、"爱死了" |
| Expressing sadness | "心里不好受"、"默默咽了" | "太难过了呜呜" |
| Expressing decisiveness | "转身就走"、"不回头" | "说走就走超飒" |
| Expressing agreement | "懂的都懂" | "太真实了！" |

### Sentence Patterns

```
✅ Use short sentences:
   "心里有本账，默默给你打分。"

✅ Use em-dashes:
   "转身比谁都快——不是不爱，只是尊严比感情更重要。"

✅ Use ellipsis:
   "那些回忆啊，就当是做了场醒不来的梦吧。"

❌ Avoid:
   "真的真的超级无敌..."
   "笑死我了哈哈哈哈"
   Multiple exclamation marks!!!
```

---

## Anti-AI Detection Tips

### 1. Add Imperfection

```
✅ "emmm怎么说呢" → casual hesitation
✅ "好像是这样" → uncertain tone
✅ "反正就是" → casual ending
```

### 2. Add Personal Perspective

```
✅ "我们射手座啊"
✅ "说实话"
✅ "其实早就"
```

### 3. Use Irregular Punctuation

```
✅ Em-dashes instead of commas
✅ Occasional missing periods
✅ "..." for pause/reflection
```

### 4. Avoid AI Patterns

```
❌ "首先...其次...最后..."
❌ "总的来说..."
❌ "综上所述..."
❌ Too-neat parallel structures
```

## Templates

### Template A: Cool Type (Recommended)

```
{星座}的{主题}，从来不是{常见误解}。

{核心表现1}。{核心表现2}，{转折或递进}。

{情感升华或留白}。
```

### Template B: Self-narration Type

```
我们{星座}啊，{核心特点}。

{具体表现1}。{具体表现2}——{解释或反转}。

{收尾感慨}。
```

### Template C: Observation Type

```
{星座}{行为/特点}，{简短描述}。

{展开1}；{展开2}；{展开3}。

{淡淡收尾}。
```

---

## Quality Checklist

After generating copy, verify:

- [ ] Title is 10-20 characters
- [ ] Total 200-300 characters
- [ ] **At least 6 paragraphs**
- [ ] Each paragraph 30-60 characters
- [ ] Each paragraph can stand alone as image text
- [ ] No exclamation mark clusters
- [ ] No "宝子们"、"姐妹们" etc.
- [ ] No "点赞收藏" calls-to-action
- [ ] Varied sentence structure (not all parallel)
- [ ] Contains em-dashes or ellipsis
- [ ] Ending has lingering impression
- [ ] Reads like human-written, not AI
