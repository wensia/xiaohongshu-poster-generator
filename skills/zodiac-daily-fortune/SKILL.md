# 星座每日运势生成器

为小红书生成星座每日运势分析文案，采用四元素 + 行星象征 + 宫位触发的多维框架。

---

## 第一步：时间节点自动判断

根据日期 `{{YYYY-MM-DD}}` 自动判断当前属于以下哪一类时间节点（可多选）：

| 节点类型 | 判断规则 | 特征 |
|----------|----------|------|
| 年初节点 | 1月1日前后 | 新起点、重置感 |
| 月初 / 月末节点 | 每月前3天 / 后3天 | 开启、收尾 |
| 季度切换节点 | 3/6/9/12月 | 阶段性转折 |
| 星座切换窗口 | 太阳换座前后3天 | 能量过渡期 |
| 普通日节点 | 无明显时间放大效应 | 日常调整 |

**使用原则**：
- 判断后，在全文语义中**自然体现节点气质**
- **不要直接点名分类**（如"今天是月末节点"）
- 节点特征影响标题关键词和段落侧重点
- 请先判断时间节点特征，再据此动态决定标题关键词、段落侧重点与整体气质

## 标题生成规则

### 基本要求

| 要素 | 要求 |
|------|------|
| 字数 | 10～20字 |
| 必含 | "射手"或具体日期 |
| 关键词 | 根据时间节点自动选择，**不固定使用"年末"** |
| 句式 | 标题使用陈述/提示/对比式，不用疑问句 |
| 节点 | 若日期触发关键节点，标题需写出月份（如“12月”“1月”） |

### 关键词风格参考（自动判断选用）

| 时间节点 | 推荐关键词 |
|----------|------------|
| 时间节点强时（年初/季度/月初月末） | 觉醒 / 校准 / 临界点 / 转向 |
| 普通日 | 提醒 / 看清 / 调整 / 筛选 |
| 星座切换期 | 重启 / 切换 / 过渡 |
| 月末 / 季末 | 收回 / 总结 / 清理 |

### 标题关键词优化（基于已分析爆文）

- 高频构件：垂类关键词（运势/提醒/状态/走向）+ 时间锚点（日期/月份）
- 触发方式：轻度好奇或对比（如“你以为/其实/原来”类语气）
- 可选结构：数字/时间点、主题直陈、对比式短句
- 使用边界：极致称号可用但要克制，避免夸张承诺

### 标题示例

```
✅ 12月末｜射手座：火焰收回，重新校准方向
✅ 射手今日提醒：不是所有方向都值得全力奔跑
✅ 1.15｜射手：看清哪些热情是惯性
✅ 射手3.21｜能量切换期，慢一步不是退步

❌ 射手座今天超级幸运！！！
❌ 必看！射手座重大转运提示！
❌ 年末射手运势（如果不是年末不要用"年末"）
```

## 张力增强技巧（从爆文提炼，择要使用）

- 标题：垂类关键词 + 时间/节点动作词 + 轻度好奇或对比（克制、不夸张）
- 开头钩子：第一句用观点式或短句结论，先定调再解释
- 段落节奏：用短句推进，形成“判断 → 解释 → 微调动作”的三步
- 情绪价值：给出读者可对号入座的感受句（如“不是你想多了”）
- 互动引导：末段留一句轻问句或自测式提示，不强求留言

## 正文结构：四元素动态框架

正文共5段，每段50～100字。

以**四元素**为结构骨架，但**不固定主题词**，根据当日星象、行星与时间节点，**动态判断每一段的侧重点**。

---

### 第一段｜火元素

| 维度 | 内容 |
|------|------|
| 关注领域 | 行动力、动力、方向感 |
| 关联行星 | 太阳 / 火星 |
| 可能主题 | 启动 / 降速 / 回收 / 冲动 / 定向 |

**写作要点**：
- 根据当日能量判断：是需要启动、还是需要收敛
- 太阳象征：核心自我、真正想要的
- 火星象征：那股想冲的冲动
- 主题由当日状态决定，不固定为"回收"

---

### 第二段｜水元素

| 维度 | 内容 |
|------|------|
| 关注领域 | 情绪、关系、感受 |
| 关联行星 | 月亮 |
| 隐性意象 | 塔罗圣杯（情感容器） |
| 可能主题 | 放大 / 去魅 / 情绪清理 / 情感确认 |

**写作要点**：
- 月亮象征：身体的信号、说不清的感觉
- 圣杯意象：情绪的容量、承载与溢出
- 主题可能是敏感度提升，也可能是情绪稳定
- 适度加入对号入座式共鸣句，但不要夸张

---

### 第三段｜土元素

| 维度 | 内容 |
|------|------|
| 关注领域 | 现实、责任、时间、金钱 |
| 关联行星 | 土星 |
| 关联结构 | 社会结构、规则、边界 |
| 可能主题 | 压力显形 / 结构调整 / 成本意识 / 稳定需求 |

**写作要点**：
- 土星象征：逃不掉的现实、该面对的账
- 不是制造焦虑，而是「承认现实反而轻松」
- 主题可能是面对压力，也可能是寻求稳定

---

### 第四段｜风元素

| 维度 | 内容 |
|------|------|
| 关注领域 | 沟通、边界、自由感 |
| 关联行星 | 水星 |
| 关联领域 | 关系、合作、信息交流 |
| 可能主题 | 筛选 / 断舍离 / 表达修正 / 自由恢复 |

**写作要点**：
- 水星象征：脑子里的声音、该说没说的话
- 风元素 = 流动、自由、呼吸空间
- 主题可能是设边界，也可能是恢复沟通

---

### 第五段｜整合段

| 维度 | 内容 |
|------|------|
| 核心要求 | **必须提及【第9宫】**（人生方向、意义） |
| 内容焦点 | 综合四元素的当前主次关系 |
| 输出方向 | 给出「调整方向」，而非结论或许愿 |

**写作要点**：
- 必须提及【第9宫】，再用生活化语言展开方向/意义
- 第9宫不必每天强调"远方"
- 但只要涉及方向、意义、人生路径，必须隐性或显性呼应第9宫主题
- 结尾是**调整方向**，不是预测结果
- 可给具体可执行的小事

## 行星使用规则

| 规则 | 说明 |
|------|------|
| 数量 | 每篇选择 2～4 颗行星作为主导 |
| 分布 | **不需要平均使用所有行星** |
| 选择依据 | 根据当日氛围决定 |
| 表达方式 | 用「生活化语言」，不使用专业术语 |

### 行星象征对照表

| 行星 | 象征领域 | 隐性表达方式 |
|------|----------|--------------|
| 太阳 | 自我、目标、方向感 | "你内心真正想要的"、"核心动力" |
| 月亮 | 情绪、直觉、内在需求 | "身体的信号"、"那些说不清的感觉" |
| 水星 | 沟通、思维、信息 | "脑子里的声音"、"那些该说没说的话" |
| 火星 | 行动力、冲动、愤怒 | "想冲的冲动"、"心里那股火" |
| 木星 | 扩张、信念、意义 | "更大的可能"、"你相信的那些事" |
| 土星 | 责任、限制、时间 | "逃不掉的现实"、"该面对的账" |

## 第9宫使用规则

| 规则 | 说明 |
|------|------|
| 必要性 | **整合段必须出现“第9宫”字样** |
| 表达方式 | 点名第9宫，再用生活化语言解释方向/意义 |
| 不必每天强调 | "远方"、"旅行"等字面意义 |
| 核心领域 | 方向、意义、人生路径、信念、视野 |

**第9宫表达示例**：
```
✅ "第9宫提醒你，方向需要被校准"
✅ "把第9宫指向的路再看清一点"
✅ "第9宫的意义感不必急着放大"
✅ "第9宫关心的是你真正想走的路"

❌ "你的第九宫被激活"
❌ "适合出远门"（除非真的相关）
```

## 内容规则

### 必须遵守

- 强调「看清—调整—校准方向」
- 行星用生活化语言表达，**不堆术语**
- 宫位只点名【第9宫】，其余不直接说
- 不鸡汤、不预测结果
- 语言有代入感，像在跟读者对话
- 每段自然过渡，元素之间有呼应
- 段落主题**动态判断**，不机械套用
- 融合：四元素流动 + 行星影响 + 塔罗象征（圣杯情绪线索）+ 宫位结构（重点第9宫）

### 严格禁止

```
❌ 具体行星度数或专业术语（如"水星逆行在射手座15度"）
❌ 直接说其他宫位名称（第9宫除外）
❌ 直接许愿或结果预测（如"今天会遇到真爱"）
❌ 煽动焦虑的表达（如"小心破财！注意烂桃花！"）
❌ 过度乐观的保证（如"一切都会变好的！"）
❌ 给宏大目标（如"今天适合规划人生方向"）
❌ 固定使用"年末"等与当前日期不符的时间词
❌ 鸡汤式结尾
❌ 堆砌术语
```

## 完整示例

### 示例1：月末节点（12月30日）

```markdown
**12月末｜射手座：火焰收回，重新校准方向**

今天火元素进入回收期。你可能感觉冲劲没那么足，目标也有点模糊——这不是退步。是身体在问你：这股火，到底是为了什么在烧？有些热情是惯性，有些才是真心。不急着往前冲，先确认一下，你内心真正想要的是什么。

水元素被放大，情绪比平时敏感。那些你以为已经放下的，可能会突然浮上来。不用强撑理性——身体比头脑诚实。如果某段关系让你隐隐不舒服，那不是你想多了，是你终于看清了。情绪不是用来透支的，是用来提醒你的。

土元素在施压。账单、deadline、那些一直拖着没处理的事——今天适合清点。不是给自己加压，是承认：有些事拖着不会消失。但也有些事，不是面对的问题，是该止损了。分清楚哪些值得继续扛，哪些该放手。

风元素今天适合做减法。你维持的关系、回复的消息、承担的期待——有哪些让你喘不过气？不是所有人都值得解释，不是所有关系都需要维护。那些让你感到压迫的互动模式，今天可以开始设边界了。

综合来看，今天的能量不适合扩张，适合收拢。把注意力从四面八方收回来，对准第9宫指向的方向。如果只能做一件事，就选那件你一直在绕开的——做完了，今天就算赢了。

#射手座 #每日运势 #星座运势 #四元素
```

### 示例2：普通日节点（1月15日）

```markdown
**1.15｜射手座：看清哪些热情只是惯性**

火元素今天处于平稳状态。不需要刻意启动什么，也不需要强行收敛。但可以问自己：最近在忙的事，是真的想做，还是做习惯了？有些动力是惯性，停下来才发现，其实早就不想继续了。

水元素偏向内省。情绪不算汹涌，但有些微妙的不舒服可能会浮现。不用急着分析原因，先承认感受存在就好。那些说不清的别扭，往往比清晰的情绪更值得留意。

土元素今天要求务实。不是大压力，但有些小事需要处理。与其让它们堆积，不如趁今天清掉几件。完成一件具体的事，比想十件大事更有用。

风元素提醒你检查沟通。有没有话想说没说？有没有消息该回没回？不是让你讨好谁，是让你清理那些卡在中间的事。该表达的表达，该沉默的沉默。

今天没有大转折，是一个「微调」的日子。不需要想太远的事，把眼前这一步走踏实。第9宫指向的方向不会因为一天的快慢改变，但每一天的清醒会让那条路越来越清晰。

#射手座 #每日运势 #星座运势 #四元素
```

## 质量检查清单

生成运势后，检查以下项目：

- [ ] 时间节点已判断，并体现在标题关键词和整体气质中
- [ ] 标题 10-20 字，关键词与当前日期匹配（非固定"年末"）
- [ ] 正文共 5 段，每段 50-100 字
- [ ] 段落顺序：火 → 水 → 土 → 风 → 整合
- [ ] 每段主题是动态判断的，不是机械套用
- [ ] 水元素段落包含圣杯意象（情绪线索）
- [ ] 融入 2-4 个行星象征（隐性）
- [ ] 整合段出现“第9宫”，并呼应方向/意义
- [ ] 无专业术语（行星度数、其他宫位名称）
- [ ] 无直接预测（会遇到xxx）
- [ ] 无焦虑煽动（小心xxx！）
- [ ] 无鸡汤、无堆术语
- [ ] 强调「看清—调整—校准方向」
- [ ] 语言有代入感，像在对话
