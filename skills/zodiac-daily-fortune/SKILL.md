# 星座每日运势生成器

为小红书生成星座每日运势分析文案，采用四元素流动框架。

---

## 核心定位

| 维度 | 设定 |
|------|------|
| 主题风格 | 年末觉醒 / 自我校准 / 去魅 / 内在对齐 |
| 核心方法 | 四元素流动（火 / 水 / 风 / 土）+ 塔罗象征 + 星象压力感 |
| 情绪基调 | 清醒、诚实、不煽动焦虑 |
| 适用场景 | 小红书每日运势推送 |

---

## 结构要求

### 标题规范

- **字数**：10～20 字
- **必含元素**：日期或星座名称
- **调性**：觉醒、提醒、警示感
- **禁止**：感叹号堆砌、过度煽情

**示例：**
```
✅ 12.30｜射手座：火焰收敛，重新校准方向
✅ 射手座今日提醒：别急着往前冲
✅ 2024.12.30 射手｜内在对齐日

❌ 射手座今天超级幸运！！！
❌ 必看！射手座重大转运提示！
```

### 正文结构（5段，每段50～100字）

| 段落 | 元素 | 内容焦点 |
|------|------|----------|
| 第一段 | 火元素回收 | 行动力、热情、目标感 |
| 第二段 | 水元素放大 | 情绪、关系、直觉 |
| 第三段 | 土元素施压 | 现实、责任、金钱、结构 |
| 第四段 | 风元素筛选 | 自由、边界、沟通 |
| 第五段 | 元素重新配比 | 总结 + 调整建议 |

---

## 内容规则

### 必须遵守

- 强调「当下状态」和「能量调整方向」
- 语言具有共鸣感和代入感
- 适合小红书用户阅读习惯
- 每段自然过渡，不生硬切换

### 严格禁止

```
❌ 具体行星度数或专业术语（如"水星逆行在射手座15度"）
❌ 直接许愿或结果预测（如"今天会遇到真爱"）
❌ 煽动焦虑的表达（如"小心破财！注意烂桃花！"）
❌ 过度乐观的保证（如"一切都会变好的！"）
```

## 各元素写作指南

### 火元素回收（第一段）

**核心问题**：今天的行动力状态如何？

```
关注点：
- 目标感是否清晰
- 热情是否需要收敛
- 冲动是否需要暂停
- 方向是否需要重新确认

语气：觉察、提醒、暂停
```

**示例：**
```
今天火元素进入回收期，你可能会感觉冲劲没那么足了。这不是退步，而是身体在提醒你：有些目标需要重新确认，有些热情需要留给更值得的事。别急着往前冲，先问问自己——这真的是我想要的吗？
```

### 水元素放大（第二段）

**核心问题**：情绪和关系层面在经历什么？

```
关注点：
- 情绪是否更敏感
- 直觉是否在提示什么
- 关系中是否有未说出口的话
- 是否需要自我照顾

语气：温柔、觉察、接纳
```

**示例：**
```
水元素今天被放大，情绪可能比平时更敏感。那些你以为已经放下的事，可能会突然冒出来。不用强行理性，给自己一点时间去感受。如果有人让你不舒服，相信你的直觉——身体比头脑更诚实。
```

### 土元素施压（第三段）

**核心问题**：现实层面有什么需要面对？

```
关注点：
- 金钱、工作、责任
- 结构性问题是否在暴露
- 是否有拖延的事需要处理
- 现实与理想的差距

语气：清醒、直面、不回避
```

**示例：**
```
土元素在施压，现实层面可能有些事需要你正视。账单、deadline、一直拖着没处理的问题——今天适合清点一下。不是要给自己压力，而是承认：有些事拖着不会消失，面对了反而轻松。
```

### 风元素筛选（第四段）

**核心问题**：信息和边界需要如何调整？

```
关注点：
- 信息是否过载
- 哪些关系需要设边界
- 沟通是否需要更直接
- 自由感从何而来

语气：轻盈、筛选、清理
```

**示例：**
```
风元素今天适合做减法。你接收的信息、你维持的关系、你承担的期待——有哪些其实可以放下？不是所有消息都需要秒回，不是所有人都需要解释。给自己留点空白，才有空间呼吸。
```

### 元素重新配比（第五段）

**核心问题**：今天的整体调整建议是什么？

```
关注点：
- 四元素如何重新平衡
- 今日最需要关注什么
- 给出可操作的小建议
- 留有余韵，不强硬收尾

语气：总结、温和、赋能
```

**示例：**
```
今天的能量配比建议：火收40%，水放30%，土稳20%，风清10%。不需要做太多，允许自己慢下来。如果只能做一件事，就选那件你一直在逃避的——做完了，今天就算赢了。
```

今天火元素进入回收期。你可能感觉冲劲没那么足，目标也有点模糊。这不是倒退，是身体在说：先停下来看看方向。那些让你犹豫的事，可能真的需要重新确认——不是所有路都值得全力奔跑。

水元素被放大，情绪比平时更敏感。一些你以为放下的人和事，可能会突然浮现。不用强撑理性，今天适合诚实面对自己。如果某段关系让你隐隐不舒服，相信直觉——身体不会骗你。

土元素在施压。账单、deadline、一直拖着没处理的事——今天适合清点。不是给自己加压，而是承认：有些事拖着不会消失。面对它，反而比逃避轻松。

风元素适合做减法。信息、关系、别人的期待——有哪些其实可以放下？不是所有消息都要秒回，不是所有人都需要解释。留点空白给自己。

今日能量配比：火收40%，水放30%，土稳20%，风清10%。不需要做太多，允许自己慢一点。如果只能做一件事，就选你一直在逃避的那件——做完了，今天就算赢了。

#射手座 #每日运势 #星座运势 #四元素 #年末校准

## 质量检查清单

生成运势后，检查以下项目：

- [ ] 标题 10-20 字，含日期或星座名
- [ ] 正文共 5 段
- [ ] 每段 50-100 字
- [ ] 段落顺序：火 → 水 → 土 → 风 → 总结
- [ ] 无专业术语（行星度数等）
- [ ] 无直接预测（会遇到xxx）
- [ ] 无焦虑煽动（小心xxx！）
- [ ] 语言有代入感，像在跟读者对话
- [ ] 结尾有调整建议，不是空洞鸡汤
- [ ] 整体读起来清醒、诚实、不浮夸
