# 8pt 网格系统详解

## 为什么使用 8pt 网格？

1. **可整除性**：8 可以被 2、4 整除，方便缩放
2. **屏幕友好**：大多数屏幕分辨率是 8 的倍数
3. **行业标准**：Google Material Design、Apple HIG 都使用 8pt 网格

## 间距规范

### 基础间距 Token

| Token | 值 | 计算 | 用途 |
|-------|-----|------|------|
| `$space-1` | 8px | 8×1 | 最小间距、图标与文字 |
| `$space-2` | 16px | 8×2 | 组内元素间距 |
| `$space-3` | 24px | 8×3 | 标准模块间距 |
| `$space-4` | 32px | 8×4 | 区块间距 |
| `$space-5` | 40px | 8×5 | 大区块间距 |
| `$space-6` | 48px | 8×6 | 主要分隔 |
| `$space-8` | 64px | 8×8 | 大留白 |
| `$space-10` | 80px | 8×10 | 安全区边距 |
| `$space-12` | 96px | 8×12 | 顶部/底部留白 |

### 常用间距场景

| 场景 | 推荐值 | 说明 |
|------|--------|------|
| 图标与文字 | 8px | 紧凑关联 |
| 同组按钮间距 | 16px | 明确分隔但保持关联 |
| 段落间距 | 24px | 视觉上独立 |
| 模块间距 | 32-48px | 明确的区域划分 |
| 安全区边距 | 80-100px | 画布边缘留白 |

## 1080×1440 画布网格

### 水平网格（12 列）

```
画布宽度: 1080px
左右边距: 80px 各边
内容区: 920px
列宽: 920/12 ≈ 76px
列间距: 16px

实际可用列宽: (920 - 11×16) / 12 ≈ 62px
```

### 垂直网格

```
画布高度: 1440px
上边距: 90px (≈ 8×11)
下边距: 90px
内容区: 1260px

行高基准: 8px
标题行高: 72px (8×9)
正文行高: 48px (8×6)
```

### 关键对齐点

| 元素 | Y 坐标 | 网格校验 |
|------|--------|----------|
| 页眉 | 90-130px | 90 = 8×11 + 2 |
| 主标题 | 200-280px | 200 = 8×25 ✓ |
| 副标题 | 320-360px | 320 = 8×40 ✓ |
| 内容起始 | 400px | 400 = 8×50 ✓ |
| 页脚分隔线 | 1350px | 1350 ≈ 8×169 |
| 页码 | 1390px | 1390 ≈ 8×174 |

## SVG 中应用网格

### 定义网格参考线（调试用）

```svg
<defs>
  <pattern id="grid8" width="8" height="8" patternUnits="userSpaceOnUse">
    <path d="M 8 0 L 0 0 0 8" fill="none" stroke="#ddd" stroke-width="0.5"/>
  </pattern>
  <pattern id="grid64" width="64" height="64" patternUnits="userSpaceOnUse">
    <path d="M 64 0 L 0 0 0 64" fill="none" stroke="#aaa" stroke-width="1"/>
  </pattern>
</defs>

<!-- 调试时显示网格 -->
<rect width="1080" height="1440" fill="url(#grid8)" opacity="0.3"/>
<rect width="1080" height="1440" fill="url(#grid64)" opacity="0.5"/>
```

### 元素坐标对齐

```svg
<!-- 好的做法：坐标落在网格上 -->
<text x="80" y="200">标题</text>        <!-- 80 = 8×10, 200 = 8×25 -->
<rect x="80" y="320" width="240" height="160"/>  <!-- 全部是 8 的倍数 -->

<!-- 避免的做法 -->
<text x="75" y="195">标题</text>        <!-- 不在网格上 -->
```

## 常见布局模板

### 居中布局

```
|←——— 80px ———→|←————— 920px —————→|←——— 80px ———→|
|   安全区     |     内容区        |    安全区    |
|              |   中心轴 x=540    |              |
```

### 左对齐布局

```
|←— 80px —→|←———————— 1000px ———————→|
| 安全区   |        内容区           |
|          | 对齐线 x=80             |
```

## 检查清单

- [ ] 所有水平间距是 8 的倍数
- [ ] 所有垂直间距是 8 的倍数
- [ ] 元素宽度是 8 的倍数（或至少是偶数）
- [ ] 元素高度是 8 的倍数（或至少是偶数）
- [ ] 安全区边距一致（左右相等，上下相等）
- [ ] 同类元素间距统一
