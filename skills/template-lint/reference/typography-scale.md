# 字号比例与层级

## 为什么需要字号比例？

统一的字号比例能够：
1. 建立清晰的视觉层级
2. 保持整体的和谐感
3. 方便维护和扩展

---

## 常用比例系统

### 1. 黄金比例 (1.618)

经典比例，适合优雅、传统的设计。

| 层级 | 字号 | 计算 |
|------|------|------|
| H1 | 72px | 44 × 1.618 |
| H2 | 44px | 28 × 1.618 |
| H3 | 28px | 17 × 1.618 |
| Body | 17px | 基准 |

### 2. 大二度 (1.333)

适合紧凑的设计，层级区分更微妙。

| 层级 | 字号 | 计算 |
|------|------|------|
| H1 | 64px | 48 × 1.333 |
| H2 | 48px | 36 × 1.333 |
| H3 | 36px | 27 × 1.333 |
| Body | 27px | 基准 |

### 3. 大三度 (1.25) - 推荐用于海报

适合内容密集的设计，层级清晰但不夸张。

| 层级 | 字号 | 计算 |
|------|------|------|
| H1 | 72px | 58 × 1.25 |
| H2 | 58px | 46 × 1.25 |
| H3 | 46px | 37 × 1.25 |
| H4 | 37px | 30 × 1.25 |
| Body | 30px | 基准 |
| Small | 24px | 30 / 1.25 |
| Caption | 20px | 24 / 1.25 |

## 字重规范

### 字重对照表

| 数值 | 名称 | 用途 |
|------|------|------|
| 300 | Light | 大字号装饰、副文案 |
| 400 | Regular | 正文、说明 |
| 500 | Medium | 副标题、强调 |
| 600 | SemiBold | 章节标题 |
| 700 | Bold | 主标题 |

### 使用原则

1. **层级递减**：标题用粗体，正文用常规
2. **最多 3 种**：一个页面最多使用 3 种字重
3. **对比明确**：相邻层级字重差 ≥100

## 中英文混排

### 字体配对

| 中文字体 | 搭配英文 | 风格 |
|----------|----------|------|
| 思源宋体 | Georgia / Times | 优雅、传统 |
| 思源黑体 | Helvetica / Arial | 现代、简洁 |
| LXGW WenKai | Georgia / Palatino | 文艺、温暖 |
| Noto Serif SC | Georgia / Crimson | 杂志、编辑风 |

### 混排行高调整

中英文混排时，行高取决于中文：

```css
/* 纯英文 */
line-height: 1.5;

/* 中英文混排 */
line-height: 1.8;  /* 中文需要更大行高 */

/* 中文标题 */
line-height: 1.3;  /* 标题可以紧凑一些 */
```

### 字距调整

| 文字类型 | letter-spacing |
|----------|----------------|
| 中文正文 | 0.05-0.1em |
| 中文标题 | 0.1-0.15em |
| 英文正文 | 0-0.02em |
| 英文大写 | 0.1-0.2em |
| 数字 | 0 |

## SVG 字体设置

### 基本设置

```svg
<text
  font-family="Noto Serif SC, Georgia, serif"
  font-size="32"
  font-weight="400"
  letter-spacing="2"
  fill="#4A3F35">
  文字内容
</text>
```

### 多行文本

```svg
<text font-family="Noto Serif SC" font-size="32">
  <tspan x="100" y="200">第一行文字</tspan>
  <tspan x="100" y="260">第二行文字</tspan>  <!-- y 差 60px = 32px × 1.875 -->
</text>
```

### 混排调整

```svg
<text y="200" font-family="Georgia" font-size="64" fill="#B86B4A">
  <tspan>85</tspan>
  <tspan font-size="48" dy="4">%</tspan>  <!-- % 符号小一号，下移对齐 -->
</text>
```
